部署 Windows Deployment Services

  2. 安装Windows Server 2019

  3. 在 Server 2019 上 安装 Deployment Services
    1. 开始菜单 》Server Manager 》 Manage 》 Roles And Features
    2. 一路 Next，选择 role： Windows Deployment Services > Add Feature > 一路 Next
    3. Role Services 要勾选 Deployment Server 和 Transport Server
  4. 在 Server 2019 上 配置使用 Deployment Services
    1. 开始菜单 》 Windows Deployment Services
    2. 邮件菜单 WDS-HL > Configure Server > 下一步 》 Standalone Server
    3. Remote Installation Folder Location 保持默认 C:\RemoteInstall
    4. PXE Server Initial Settings : Respond to all client computers(known or nknown)
    5. 下一步直到配置完成。
  5. 获取 windows image 文件
    1. 打开windows 10 iso，进入sources目录
    2. 找到 boot.wim 和 install.wim，拷贝到 C:\RemoteInstall
      1. 可以重命名下，后面好找，比如 boot.wim 重命名为 win10-202h-boot.wim
    3. 在 Deployment Server 中右击 boot images 文件夹图标，选择刚刚的boot.wim文件，并添加。
    4. 在 Deployment Server 中右击 install images 文件夹图标， 创建一个Image Group，比如win10 20h2，选择刚刚的 install.wim文件，并添加所有出现的 image，但是 不勾选 Use the default name and description for each of the selected images，后面自己取个简洁的名字。
  7. 使用 Windows Deployment Server 给 VM 安装 windows
    1. 在virtualbox上创建虚拟机，设置 System > Motherboard > Boot Order > 勾选 Network 并放置到顶部
    2. 启动虚拟机，按F12进行PXE 网络启动
    3. 弹出的用户名，填写 Server 2019上的帐号，用户名： WIN-FVLQEOJ3HLB\Administrator
      WIN-FVLQEOJ3HLB 是 Windows Deployment Server 界面中的服务器名称。

创建自定义的 windows 安装文件

  1. 使用 Windows System Image Manager 创建 answerfile （应答文件），answer file 包含了windows 10安装过程中的配置。
  2. 下载 Windows ADK 工具集合，其中包含工具 Windows System Image Manager
    ADK download for Windows 10
  3. 安装 ADK 工具集合，启动 Windows System Image Manager
  4. 文件 菜单 》新建应答文件
  5. Windows 映像 》Components 》amd64_Microsoft-Windows-Setup_10.0.19041.1_neutral
    1. UserData 》 右键菜单 》添加设置以传送1 WindowsPE（1）
    2. UserData - AcceptEula: true
    3. UserData - ProductKey - Key : 产品序列号
  6. amd64_Microsoft-Windows-Shell-Setup_10.0.19041.1_neutral
    1. OOBE - 右键菜单 - Add Setting to Pass 7 oobeSystem
      1. HideEULAPage : true
      2. HideOEMREgistrationScreen : true
      3. HideOnlineAccountScreen : true
      4. HideWirelessSetupInOOBE : true
      5. NetworkLocation: Work
      6. SkipMachineOOBE: true
      7. SkipUserOOBE: true
      8. 删除 OOBE - VMModeOptimizations
    2. UserAccounts - 右键菜单 - Add Setting to Pass 7 oobeSystem
      1. AdministratorPassword - value : 设置管理员密码
      2. 删除 DomainAccounts
      3. LocalAccounts - 右键菜单 - 插入新建 LocalAccount
        1. Group: Administrator
        2. Name: your-account-name
        3. DisplayName: your-name
        4. LocalAccount - Password 设置密码
  7. FirstLogonCommands - SynchronousCommands
    1. CommandLine: reg add "HKLM\Software\Microsoft\Windows\CurrentVersion\RunOnce" /v "disabledmwappushservice" /t REG_SZ /d "sc config dmwappushservice start= disabled"
    2. description: Disable DM WAP Push Service
    3. Order: 1
    4. RequireUserInput: false

  9. 创建的应答文件 unattend.xml，拷贝到当前运行的Windows 10 的 C:\WIndows\System32\Sysprep 下面

  10. Sysprep
    1. 双击启动Sysprep
      1. System Cleanup Action: Enter System Out-of-Box Experience(OOBE)
      2. 勾选 Generalize
      3. Shutdown Options: Shutdown
      4. 确定后会运行一段时间，然后关机
    2. 如果报错，可查看 C:\WIndows\System32\Sysprep\setupact.log
  11. VM 设置从Network 启动，启动到Image Deploy Server
  12. 看到 Windows Setup 界面后，按下 Shift + F10 进入命令行界面

 # 生成wim文件

 dism /Capture-Image /ImageFile:Win10-Pro-20H2.wim /CaptureDir:C:\ /Name:"Windows 10 Pro 20H2"
  13. 生成wim文件存放在 deploy server 的 C:\RemoteInstall 目录下
  14. 将生成的 wim 文件导入 deploy server
  15. 设置应答文件
    1. 双击这个Image，勾选 Allow image to install in unattended mode
    2. 选择 Select File… ，在弹出的文件对话框中选择目标 unattend.xml

这个方法生成install.wim 打包进 iso，安装时到专业版、教育版选项的时候，没有选项了。。。。

将当前运行的 Windows 10 生成 iso 安装文件

  1. 删除当前用户，此电脑 》管理 》用户
  2. 运行 cleanmgr ，进行临时文件清理
  3. %windir%\System32\Sysprep\sysprep.exe /audit /reboot
    1. 重启后，弹出“系统准备工具 3.14”对话框，上面2个选项：
    2. 系统清理操作：进入系统全新体验（OOBE）
    3. 勾选“通用”
    4. 关机选项：关机
  4. 确定关机
  5. 启动这个windows就进入了 欢迎设置界面。一通安装windows配置。。。
  6. 在C盘另外的盘，例如D盘，创建文件夹 Scratch
  7. 从windows安装U盘启动
  8. 启动到Windows安装的第一个界面时，按 Shift + F10 进入命令行
  9. diskpart 进入 diskpart 软件
  10. list vol 看看各个分区的盘符，例如，系统盘C现在是F，D盘现在是E
  11. dism /capture-image /imagefile:E:\install.wim /captureDir:F:\ /ScratchDir:E:\Scratch /name:"windoes10Prox64" /compress:maximum /checkintegrity /verify /bootable
  12. 执行完成，重启进入Windows之后，D盘就可以看到 install.wim 文件
  13. mount windows 原来iso文件，进入mount的光盘，把文件都拷贝出来，例如，新建个名叫 myiso 文件夹放着。
  14. 用生成的 install.wim 替换 myiso/sources/install.wim
  15. 下载 Windows ADK 工具 adksetup
    https://docs.microsoft.com/en-us/windows-hardware/get-started/adk-install
  16. 已管理员身份运行 adksetup
    1. Select the Features you want to install 页面选择： 只选 Deployment Tools
  17. 安装完成后，在开始菜单找到 Deployment and Imaging Tools，这是个命令行工具，已管理员身份运行
  18. 运行
    oscdimg.exe -m -o -u2 -udfver102 -bootdata:2#p0,e,bd:\myiso\boot\etfsboot.com#pEF,e,bd:\myiso\efi\microsoft\boot\efisys.bin d:\myiso d:\Windows10Prox64.iso
