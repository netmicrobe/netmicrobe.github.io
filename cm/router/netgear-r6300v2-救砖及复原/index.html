netgear-r6300v2-救砖及复原

March 01, 2020

  • 参考

用 TTL 线将电磁炉从楼主的 384 复原到之前的 7.9.1，拯救活动胜利完成，双程往返

在 R6300v2 从梅改 7.9.1 版升级到楼主的 RT-AC6300v2 前，务必在 SSH 命令行窗口中，使用下面的命令在路由器中备份梅改 7.9.1 版固件的 CEF 和 board_data 原厂产品数据：

~~~
# 备份CFE
cat /dev/mtd0 > /tmp/home/root/original_r6300v2_cfe.bin

# 备份board_data
cat /dev/mtd4 > /tmp/home/root/original_r6300v2_board_data.bin
~~~

备份好后，将得到的 “.bin” 备份数据文件从路由器中下载到本地电脑保存，然后再去做固件升级。

RT-AC6300v2 复原梅改 7.9.1 版 R6300v2 步骤

  1. 将电磁炉的本地地址务必改为 192.168.1.1，否则会有麻烦。
  2. 电脑网卡本地 IP 地址设为 192.168.1.100，用网线连接至电磁炉的 LAN2/LAN3 口。
  3. 准备并连接好 TTL 刷机线，以及在电脑启动 SecureCRT 客户端软件（也可以用 putty 等工具）、TFTPD 软件（不是 Windows 系统自带的 TFTP.EXE），使用方法自己找。
    32 位下载地址：http://tftpd32.jounin.net/download/tftpd32.452.zip
    64 位下载地址：http://tftpd32.jounin.net/download/tftpd64.452.zip
  4. 在电脑的 tftpd 服务器目录（或者启动后指定相应的目录也可以）里放入之前备份好的 R6300v2 原始 CFE 文件 original_r6300v2_cfe.bin 及论坛上下载的固件 R6300V2_380.70_0-X7.9.1-koolshare.trx。在 tftpd 软件窗口第二行的 Server interfaces 选择框里选择网卡的 192.168.1.100 地址。
  5. 打开 SecureCRT，按住电磁炉的复位键后接通电源，不断按 Ctrl+C，终止系统自动启动，让电磁炉处于 CFE 模式下，也就是按住 Ctrl+C 键后直到开机显示 CFE> C。
  6. CFE 中按序执行如下三条命令：（如果之前已在楼主的 384 固件系统里刷过备份的 380 固件系统的 CFE，则忽略此步操作，直接进入下一步） 
    1
2
3
     nvram erase
 flash -noheader 192.168.1.100:original_r6300v2_cfe.bin nflash0
 reboot
  7. 系统重启后，再次让电磁炉进入 CFE 状态，然后在 CFE 中按序执行如下三条命令： 
    1
2
3
     flash -noheader 192.168.1.100:R6300V2_380.70_0-X7.9.1-koolshare.trx nflash0.trx
 nvram erase
 reboot
  8. 系统再次重启，SecureCRT 中的数据刷啦啦刷啦拉的跑，你的电磁炉至此重重获新生中的新生。。。
  9. 如果之前做过 board_data 原厂产品数据备份，在路由器重启后，首先开启 SSH 功能，使用 SSH 终端连接路由器，按序执行如下三条命令: 
    1
2
3
     nvram erase
 ./mtd-write -i original_r6300v2_board_data.bin -d board_data
 reboot
  10. 系统重启后，电磁炉可彻底复原升级前状态。
  11. 如果之前没有做过 board_data 原厂产品数据备份，可在电磁炉底部的设备标牌里找到 MAC 地址（假设为 ABCDEFGHIJKL），使用 SSH 终端连接路由器，字母小写方式输入并按序执行如下三条命令: 
    1
2
3
     nvram set et0macaddr=ab:cd:ef:gh:ij:kl
 nvram commit
 reboot
  12. 电磁炉重启后可在梅改 7.9.1 版固件下使用，与第 9 步的区别是系统中缺少部分原厂产品数据，但不影响路由器使用。

  13. PS：若想刷回网件原厂固件，只需在 arm380 的 7.9.1 固件已正常运行的基础上刷返回固件，然后在原厂系统状态下，使用 TTL 或 Telnet 方式连接电磁炉，用楼下 @dexi2009 提供的命令烧回网件主要的原厂产品数据，即可复原国行电磁炉原版状态，最后可刷 “.chk” 后缀的所希望版本的原厂固件。
