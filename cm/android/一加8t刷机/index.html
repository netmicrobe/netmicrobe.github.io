总结

一加8T 氢OS在某个版本开始，可以安装和浏览 Play Store，但是无法下载，一直显示下载中。。。

可以刷氧OS来解决：

  1. 下载 氧OS 全量包，例如，OnePlus8TOxygen_15.O.29_OTA_0290_all_2110091944_3d8159dff77f4ef1.zip
  2. 将zip包拷贝到根目录
  3. 设置》系统》系统更新》本地升级》选择刚拷贝进手机的氧OS全量zip包
  4. 升级完成重启
  5. 原来氢OS安装的软件、指纹等设置都还在

    5. 刷完后，自动进入系统

EDL 模式

EDL / Emergency Download Mode，独立于 bootloader， 通过 Firehose 或 Sahara 协议来强制刷写固件。

进入EDL模式的方法：

  1. 从系统进入： 执行 adb reboot edl
  2. 从关机状态进入： 同时按住音量 ：加减（不要按开机键）， 手机插上数据线
  3. 从fastboot进入，研究了下目前没成功
    • 方法一，修改 fastboot 可执行文件，来支持 reboot 进 EDL
      参考 How to reboot to EDL from fastboot 要修改 fastboot 可执行文件。
      下载了帖子里改好的 fastboot_edl.exe，在 bootloader 模式下执行 fastboot_edl.exe reboot-edl，并没有进入EDL模式，而是重启进系统了。
  • 方法二，看 oem 本身是不是支持。
    参考 What is EDL Mode? How to Enter EDL Mode on Any Qualcomm Device

从 ColorOS 12.1 / KB2000_11_C.28 降级到 Hydrongen OS 11.0.12.12.KB05

  1. ColorOS 》 设置 》关于本机 》点击“版本号”多次，开启开发者模式，才能使用本地刷OS包
  2. 本地刷包： 设置 》关于本机 》点击版本 》右上角三个点菜单 》本地安装
  3. 选中拷贝到根目录的 OnePlus8THydrogen_15.H.31_OTA_0310_all_2110102354_downgrade_for_ColorOS_99349128626740e3.zip
  4. 点击“立即解压”，提示 将会清空所有数据，点击“已备份、继续”
  5. 解压十多分钟后，还要点击下“立即安装”
  6. 启动后，系统版本恢复到氢OS： Hydrongen OS 11.0.12.12.KB05
  8. 据说：降级后由于数据清空将导致部分系统自带基础应用丢失，请务必按照以下步骤下载并安装全量包！
  10. 拷贝OnePlus8THydrogen_15.H.31_OTA_0310_all_2110102354_c1fc5e100ad14e1d.zip到根目录
  11. 氢OS 设置 》系统 》 系统更新 》右上角齿轮图标 》本地升级
  12. 选中OnePlus8THydrogen_15.H.31_OTA_0310_all_2110102354_c1fc5e100ad14e1d.zip 》立即升级
  13. 升级发现 确实多了很多app，是不是系统自带基础应用丢失，不确定，很多预置第三方app都进来了。
  14. — — — 再升级回 ColorOS
  15. 方法一： 系统自动下载ColorOS的升级包，安装后，升级到 KB2000_11_C.07，然后提示下载 ColorOS 12.1 / KB2000_11_C.28
  16. 方法二： 手动刷
  17. ColorOS 12 C.07 / KB2000_11_C.07 / 6016ebd34f1c4646b2ba5bb68c4b4cdf.zip 拷贝到根目录
    氢OS 设置 》系统 》 系统更新 》右上角齿轮图标 》本地升级
  18. ColorOS 12.1 C.28 / KB2000_11_C.28 / b26ad2c2a2504967902ec2eb69b6c472.zip 拷贝到根目录

选择“高级选项”选择“启动设置”。点击“重启”按提示输入“7”禁用驱动程序强制签名。）
