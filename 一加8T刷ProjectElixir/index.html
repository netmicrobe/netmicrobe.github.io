从 evolution X 7.6.8c1 刷到 ProjectElixir 3.6

  1. 进入 evolution X 的 recovery
  2. Recovery 中恢复下出厂设置： Factory Reset -> Format data / factory reset
  3. sideload the ProjectElixir 3.6 .zip package

Installation Guide For Project Elixir on Kebab

From version 3.4
Based on OOS13 firmware
Will only work for Oneplus 8T(9R builds are given seperately)

Firmware Instructions:

  1. Method 1: If from OOS11 or OOS12

    1. Unlock bootloader
    2. 下载 OOS13 fw ： 下载链接
    3. It will flash fw after detecting the ddr type so don’t worry about bricking the device
    4. Flash the fw using this recovery
    5. Follow from Method 2

  2. Method 2: If from OOS13

    1. Unlock bootloader
    2. Download the rom and flash via this recovery
    3. Follow the clean flash instructions given below

  3. Clean Flash:

    1. Download the latest build
    2. Take a backup for safe side
    3. Boot to Recovery of your choice
    4. Flash or Sideload the rom
    5. Go back to recovery and select Format Data
    6. Reboot

  4. DIRTY FLASH [A13 to A13]

    1. Boot to recovery
    2. Flash or Sideload the rom
    3. Reboot
